

\# Guitar Mode Trainer



A lightweight, mobile-first web application designed for guitarists to practice scales, modes, and fretboard visualization under pressure. 



The app generates a completely randomized sequence of all \*\*98 unique combinations\*\* of strings, root notes, and modes, forcing you to think fast and react immediately.



🔗 \*\*\[Try the Web App Live Here](https://disturbingthefield.github.io/Guitar-Mode-Trainer/)\*\*



\---



\## Features



\* \*\*98 Unique Combinations:\*\* Loops through all 7 musical keys (A, B, C, D, E, F, G) across all 7 modes on both the \*\*E\*\* and \*\*A\*\* strings.

\* \*\*True Randomization:\*\* Every loop of 98 combinations is completely shuffled. You will never get the same combination twice in one round.

\* \*\*Adjustable Timer:\*\* Set your own pace from 1 second up to 5 minutes (300 seconds).

\* \*\*Live Countdown:\*\* See exactly how many seconds you have left before the next combination hits.

\* \*\*Loop Progress Tracker:\*\* Keeps track of your workout session (e.g., `Remaining: 95 / 98`).

\* \*\*Mobile-First \& Scalable:\*\* Designed with a vertical layout perfect for smartphone stands. Includes 10 different font sizes so you can easily read the screen from a distance while holding your guitar.

\## How to Practice



1\. \*\*Set your time:\*\* Enter the number of seconds you want each prompt to display (e.g., `5` or `10` seconds).

2\. \*\*Choose text size:\*\* Adjust the text size so it is easily readable from your practice position.

3\. \*\*Press START:\*\* The app will instantly display a prompt using a clean, shorthand format:

&#x20;  \* `E: G Mix` $\\rightarrow$ Play \*\*G Mixolydian\*\* starting from the \*\*E-string\*\*.

&#x20;  \* `A: D Aeo` $\\rightarrow$ Play \*\*D Aeolian (Natural Minor)\*\* starting from the \*\*A-string\*\*.

4\. \*\*React Fast:\*\* There is no "Next Up" preview. You have to locate the root note and mentally map out the intervals instantly.

5\. \*\*Press STOP:\*\* Pauses the routine and resets the loop.
